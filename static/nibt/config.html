config module

class config.Config(*, hard_reset=False, config_filename='config.yml', bundle_config_file='config.yml')

Bases: dict

Creates the configuration dict to allow user different configurations.

Parameters

  • hard_reset (Bool, optional) – Set True to reintialise the user configuration folder, defaults to True

  • bundle_config_file (str, optional) – Original factory default name of config file name, defaults to ‘config.yml’

  • config_filename (str, optional) – Config file intended to be loaded by user, defaults to ‘config.yml’

Returns

configurations loaded from user configuration file config.yml

Return type

dict

clear() → None.  Remove all items from D.
copy() → a shallow copy of D
factory_reset()

Backups the entire working folder by renaming the user working dir Re-initialise by creating new configuration files and foldrs from factory defaults

items() → a set-like object providing a view on D's items
keys() → a set-like object providing a view on D's keys
load_yaml(filepath: Path = 'config.yml')

loads the yaml files

paths_init(hard_reset=False)

Initialises path read from the config.yml into pathlib.Path Creates the working directories in user’s working directory, defaults to My Documents/tetk Creates the configuration folder in user’s working directory, defaults to My Documents/tetk/_configurations Copies factory default config files to user configuration folder

Parameters

hard_reset (str) – Set True to delete the user configuration folder then restore using factory defaults, defaults to True

Returns

None

pop(k[, d]) → v, remove specified key and return the corresponding value.

If key is not found, d is returned if given, otherwise KeyError is raised

setup_logger(name='nibt', default_loglevel=20)

Function to initialise logging function. Logs to both console stdout and also a log file Log file will be logged to user working directory, defaults to My Documents/_TOOLS_tetk/tetk.log

Parameters

name (str) – Name of the log file, f”{name}.log”

Returns

None

update([E, ]**F) → None.  Update D from dict/iterable E and F.

If E is present and has a .keys() method, then does: for k in E: D[k] = E[k] If E is present and lacks a .keys() method, then does: for k, v in E: D[k] = v In either case, this is followed by: for k in F: D[k] = F[k]
