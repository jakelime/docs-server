class config.Config(*, hard_reset=False, config_filename='config.yml', bundle_config_file='config.yml')

Bases: dict

Creates the configuration dict to allow user different configurations.

Parameters

  • hard_reset (Bool, optional) – Set True to reintialise the user configuration folder, defaults to True

  • bundle_config_file (str, optional) – Original factory default name of config file name, defaults to ‘config.yml’

  • config_filename (str, optional) – Config file intended to be loaded by user, defaults to ‘config.yml’

Returns

configurations loaded from user configuration file config.yml

update([E, ]**F) → None.  Update D from dict/iterable E and F.

If E is present and has a .keys() method, then does: for k in E: D[k] = E[k] If E is present and lacks a .keys() method, then does: for k, v in E: D[k] = v In either case, this is followed by: for k in F: D[k] = F[k]
