Triggers the ‘end lot’ sequence

  1. Generates test datalog (single CSV file) from list of data files in the working folder

  2. Archives the data files from the working folder to archive folder

3. Remove interim files (single-DUT CSV reports), using overflow concept. If > N, old files will be deleted

Parameters

  • print_computation (bool, optional) – flag for TestDataObject.to_pdSeries(), defaults to False

  • archive_raw_files (bool, optional) – flag to archive tdms files found in the working folder, defaults to True

  • remove_csv_reports (bool, optional) – flag to remove unnecessary single DUT report CSV files, defaults to True
